# ⚙️ Installation

Installing Oraxen is very simple. All you need to do is put `oraxen-version.jar` inside your plugins folder and restart your server!\
Oraxen also depends on [ProtocolLib](https://ci.dmulloy2.net/job/ProtocolLib/lastSuccessfulBuild/artifact/build/libs/ProtocolLib.jar), so make sure you also have this plugin!
